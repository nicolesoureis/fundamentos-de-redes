# Projeto 001 - Entendendo o Tráfego de Rede

## Objetivo
Compreender como funciona o tráfego de rede em nível básico, analisando a comunicação entre um dispositivo e a internet em ambiente doméstico.

## Ferramentas utilizadas
- Prompt de Comando (Windows)
- ipconfig
- ping
- nslookup
- netstat

## Conceitos aplicados
- Modelo OSI
- Modelo TCP/IP
- Endereçamento IPv4
- Gateway
- DNS
- ICMP
- Portas e estados de conexão

## Metodologia
Foram executados comandos básicos de rede para observar:
- Configuração local do dispositivo
- Comunicação com um host externo
- Processo de resolução de nomes (DNS)
- Conexões e portas abertas na máquina

## Observações
Os testes foram realizados em ambiente doméstico, sem exposição de informações sensíveis ou corporativas.

## Conclusão
Este laboratório permitiu compreender como o tráfego de rede ocorre na prática, conectando conceitos teóricos (OSI e TCP/IP) com observações reais do sistema operacional.
